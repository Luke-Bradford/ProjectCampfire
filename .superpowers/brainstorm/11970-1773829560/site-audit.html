ProjectCampfire — Site Audit

What we've built, what's working, and where the gaps are

● Working well ● Exists but needs polish ● Missing or incomplete

◑ Navigation & Shell

  • Three-column layout, mobile bottom nav
  • Friends tucked away — not always visible
  • No persistent "who's online" sidebar
  • No quick-jump to a friend or DM from anywhere

◑ Availability / Scheduling

  • Weekly template + overrides model
  • Custom grid (drag, resize, overnight)
  • Loads showing blank — current time not in view
  • No "jump to now" or smart scroll on load
  • Group overlap view exists but feels buried

◑ Profile

  • Avatar, bio, stats, games tab, posts tab
  • Steam integration, activity card
  • No "gamer card" feel — reads more like settings
  • Now playing / currently online status low-key
  • No favourite games pinned to profile

◑ Groups

  • Grid layout, colour strips, member count
  • Events, polls, discussion per group
  • Group "home" feels like a settings page
  • No sense of group identity / activity at a glance
  • No pinned post or group bio/banner

◑ Friends

  • Friends list, Steam suggestions, invite link
  • Online/offline status dot
  • Friends page feels passive — just a list
  • No "what are my friends doing right now" section
  • No persistent friends panel/rail on desktop

● Events & Planning

  • Timeline view, RSVP, polls
  • Game voting, co-host, event discussion
  • Right panel shows upcoming events on desktop
  • Creating an event still requires several steps

● Feed

  • Infinite scroll, group/friend filters
  • Hot vs New sort, GIF picker
  • Feed is the homepage — but feels generic
  • No "at a glance" dashboard panel above feed

● Games Library

  • Grid/list, sort, filter, platform badges
  • Steam sync, IGDB catalog search
  • Group ownership view on game detail

Key themes to discuss

  1. Availability smart scroll — land the user where their day is, not at midnight
  2. Persistent friends rail — always see who's online without navigating away
  3. Gamer profile identity — make the profile feel like a gaming card, not a form
  4. Group home as a hub — lead with activity, not configuration
  5. Dashboard widget above feed — quick glance at what needs your attention today
