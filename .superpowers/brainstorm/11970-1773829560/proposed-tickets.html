Proposed UX Improvement Tickets

Iterative improvements across 5 areas — click any you'd like to discuss or change

Availability
UX-01 Scroll availability grid to current time on load
When the weekly schedule page loads, auto-scroll to the current hour so the user lands where their day is rather than at midnight. Add a "now" indicator line. Works for both early risers and night owls.
Quick win UI
UX-02 Add "Jump to now" button on availability calendar
A small sticky button that scrolls the grid back to the current time whenever the user has scrolled away. Common pattern in calendar apps.
Quick win UI
Right Panel / "What's Happening" Sidebar
UX-03 Expand right panel into a full "What's Happening" sidebar
Extend the existing right panel (desktop only) beyond upcoming events to include: active polls needing your vote, recently closed polls, and an online friends widget. Scrollable, grouped by section. Replaces the current bare events list.
Medium UI
UX-04 Online friends widget in right panel
Show friends who are online or recently active. Display their avatar, name, and current game if playing. Clicking navigates to their profile. This is the Discord-style "always know who's around" presence feel.
Medium UI Data
Profile — Gamer Identity
UX-05 Redesign profile header as a gamer identity card
Replace the current plain header (avatar + name + bio) with a richer identity card: large avatar, display name, username, bio, Steam veteran date, stat row (games owned, friends, groups, total playtime), online/now-playing status. Should feel like a gaming card, not a settings form.
Medium UI
UX-06 Pinned favourite games on profile
Let users pin up to 4–6 favourite games to their profile. Shows cover art, total playtime, and achievement count. Visible to friends. Backed by a simple favourites flag on the game ownership record.
Medium Data UI
UX-07 Surface Steam achievements on profile and game detail
Fetch and cache achievement counts per game from the Steam API. Show total achievements unlocked on the profile stat row and per-game on the game detail page. No individual achievement listing needed — just counts as a prestige signal.
Larger Data
Groups — Hub Feel
UX-08 Add group command centre section at top of group page
Lead the group page with a compact hub: next session (date, RSVP status, who's going), active poll needing your vote, and a members-online strip. Move settings and invite link lower. The group should feel active from the moment you land on it.
Medium UI
UX-09 Show online members with current game in group member list
The data is already there (current game name per member) but the member list doesn't make it visually prominent. Surface it: green dot + "Playing X" inline. Makes the group feel alive even when there's no active event.
Quick win UI
Navigation
UX-10 Add Friends to mobile bottom nav
Friends is currently only accessible via the mobile sidebar menu. Given it's a primary social object, it should be one tap away. Swap or add it to the bottom nav bar — possibly replacing the least-used current item.
Quick win UI
